LolaBot Embed Integration Test

This page demonstrates how the chat widget appears on an external website.

Integration Instructions

To add this chat to your website, simply include the following script tag before the closing </body> tag: 

                <script 
                src="http://localhost:3000/embed/lolabot.js"
                data-bot-id="lola-demo"
                data-user-id="lola-demo-user"
                data-theme="light"
                ></script>
